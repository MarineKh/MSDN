Naming Styles

There are no naming stytles defined for XML elements.But here are some commonly used:

Style Example Description
Lower case firstname all letters are in lower case
Upper case FIRSTNAME ALL LETTERS ARE IN UPPER CASE
Underscore first_name Underscore_separates_words
Pascal case FirstName Uppercase Is Each First Letter In Each Word
Camel case firstName uppercase Is Each First Letter In Each Word Except The First

If you choose a naming style, it is good to be consistent!

XML document often has a correspnding database. A good practice is to use the naming rules of your database for the

elements in the XML documents.

There are 5 predefined entity references in XML:

&lt; < less than
&gt; > greater than
&amp; & ampersand
&apos; ' apostrophe
&quot; " quotation mark
This box has rounded corners!

Note:Only the characters "<" and "&" are strictly illegal to be used in XML. The greater than character is legal, but it is a good habit to replace it.
